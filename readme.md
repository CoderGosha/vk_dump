Vk dump
-------
Install:
```
pip3 install -r requirements.txt 
```
Usege:
```
login with password:
	python3 vk_dump.py -vl <vk_login> -vp <vk_password>
```
```
login with token:
	python3 vk_dump.py -vt <vk_access_token>
```
```
dump history of specific account:
	vk_dump.py -id <id_of_account>
```
```
dump count users:
	vk_dump.py -c 10
```
```
dump photo:
	vk_dump.py -dp 1
```
